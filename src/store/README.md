Здесь реализован стор из библиотеки Redux toolkit
Создан он для хранение данных, в дальнейшем с помощью него можно будет проверять обновилась ли данные на сервере по хешу.

store.ts - стор
selector.ts - селектор из стора
thunk.ts - апи для взаимодействия с mockAPI
chartSlice.ts - редусер из редукс туллкит
